This android application is intend to vary the control variables for greenhouse automation 
via bluetooth. Sliders used for this and in the meantime user can see the different parameters
of greenhouse such as temperature, humidity etc. App does not control the greenhouse elements 
directly, just changes the threshold values to the desired value. 

.apk is an android application file. Can be downloaded and used on an android device. 
.aia is a MIT Appinventor file of app that can be opened and modified in the MIT Appinventor 
website for future projects.
